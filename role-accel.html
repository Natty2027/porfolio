Tech Role Accelerator

A premium AI-powered career simulator that bridges the gap between learning and employment. Dynamic learning paths, audio-enabled mock interviews, resume tailoring — all behind a Stripe-backed Pro tier.

Live demo →
React 19 Vite Tailwind v4 Google Gemini 1.5 Clerk Neon Postgres Stripe Vercel

The idea

Most career-prep tools tell you what to study. Role Accelerator drops you into a simulated workday for the role you're targeting — answering on-call tickets, code-reviewing teammates, defending a system design choice in front of a fake exec. You fail safely and improve fast.

What I built

  • Dynamic learning paths — Gemini-generated roadmaps tailored per role. Real-time skill trending via Google Search grounding so the recommendations don't go stale.
  • Interactive job simulator — high-stakes workplace scenarios with industry-standard feedback. Resume tailor does line-by-line AI analysis. Concept arcade is a gamified quiz layer with dynamic difficulty.
  • Mock interview with audio — audio-enabled interviews scored in real time. The grader is also Gemini, with a custom rubric per role.
  • Production SaaS infrastructure — Stripe Checkout + Customer Portal + signed webhooks, a withPro entitlement helper gating five premium AI endpoints, public IP-rate-limited demo on the landing page, server-side heart refill for free users, admin override via env var.
  • Auth + persistence — Clerk for sessions, Neon Postgres for users / progress / demo usage. Vercel serverless functions in /api with SPA rewrite for the React app.

The "wow"

The free demo on the landing page is a working AI bullet-point rewriter — anyone can paste a resume line and see it improved in real time, no sign-up. Conversion to signed-up users on a small budget got the project past the "looks cool but does it actually work" sniff test.
