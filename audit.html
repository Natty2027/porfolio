C-P Unified Audit

Institutional-grade security auditing platform built for small and mid-size enterprises. Combines traditional audits with a Gemini-powered threat simulator and real-time global intelligence search.

React 18 Node.js Express SQLite Google Gemini OAuth 2.0 JWT Recharts Vitest

The idea

Small companies usually fall into one of two security postures: pay an audit firm $50k a quarter, or wing it. C-P Unified Audit aims at the middle — a self-serve auditing dashboard that scores you across five domains and tells you what to fix first, plus an AI red team that walks through attack scenarios on your actual stack.

What I built

  • Multi-domain scoring dashboard — evaluates security posture across Physical Access, Network Security, Personnel Training, Data Privacy, and Compliance. Custom radar-chart visualization (Recharts) and metric cards with trend indicators.
  • AI threat simulator — input a threat type, target asset, and existing controls, and Gemini runs a high-fidelity attack simulation with mitigation recommendations rendered as Markdown reports.
  • Federated intelligence search — Gemini grounded against Google Search to surface real-time CVE data, NIST standards, and trending industry threats. Distills technical data into executive summaries.
  • Longitudinal trends — interactive line charts track audit performance over time. Automated direction detection (Positive / Negative) based on history.
  • Security-first backend — Node/Express with strict CORS, JWT validation, OAuth 2.0 with Google and GitHub. Better-sqlite3 for reliable isolated storage. Vitest + Supertest covering the protected security routes for CI/CD.

The "wow"

The threat simulator outputs are good enough that internal security teams have used them as starting drafts for their own incident reports. Grounded RAG via Google Search means responses cite real CVEs — not hallucinated ones.
