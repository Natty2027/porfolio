Cognivate

An AI study app that turns any document into chapters, vocab, and quizzes. Currently in App Review for the iOS App Store.

React Native Expo TypeScript EAS Build Claude API Clerk RevenueCat Sentry

The idea

Students upload PDFs, photos of handwritten notes, and pasted text. Cognivate uses Claude to turn that messy input into structured study modules — readable chapters, vocab cards for spaced repetition, and adaptive quiz questions. Plus an audio tutor mode for hands-free review.

What I built

  • Generation pipeline — accepts noisy student uploads and produces consistently-structured study modules. Structured output, retry-on-parse-error, length-aware chunking for long materials, graceful fallbacks for insufficient input.
  • Subscription infrastructure — Apple in-app subscriptions through RevenueCat with server-to-server receipt validation via App Store Connect API key. Entitlement-driven feature gates throughout the app, sandbox testing pipeline, and a hardened paywall that handles purchase cancellation, restoration, and post-expiration UX.
  • Custom-domain auth — Clerk on clerk.cognivate.co with custom OAuth credentials so Google's consent screen shows my branding. Native-app SSO deep-link handling, allow-listed redirect URLs, post-sign-in entitlement sync to RevenueCat.
  • EAS Build release pipeline — single Expo project ships two app identities. Public TestFlight beta (com.cognivate.mobile) and a v2-preview variant (com.cognivate.mobile.v2) for in-flight features. Selected at build time via APP_VARIANT so iOS treats them as separate apps.

What went wrong (and got fixed)

Apple rejected the first submission with four issues: ATT/tracking labels, a Google sign-in redirect URL mismatch, an authentication-code blocker on the demo account, and a missing expired-subscription demo state. Fix walked through every dashboard: App Store Connect privacy labels, Clerk's mobile SSO allowlist, App Store Connect API key for RevenueCat receipt validation, and a sandbox subscription cycle to demonstrate the expired state.

Status

Build 27 in App Review. Real RC sandbox purchases verified end-to-end. TestFlight public-beta link coming after first approval.
